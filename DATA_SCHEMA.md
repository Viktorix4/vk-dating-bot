# Схема данных: favorites.json

Файл `favorites.json` хранит список пользователей, добавленных в «Избранное».

## Формат записи

Каждая запись — это объект (словарь) со следующими полями:

| Поле          | Тип     | Описание |
|---------------|---------|----------|
| `vk_id`       | int     | Уникальный ID пользователя ВКонтакте |
| `first_name`  | str     | Имя пользователя |
| `last_name`   | str     | Фамилия пользователя |
| `profile_url` | str     | Ссылка на профиль вида `https://vk.com/id{vk_id}` |
| `photos`      | list[str] | Список из 3 attachment-строк фотографий (формат: `photo{owner_id}_{media_id}`) |
| `saved_at`    | str     | Дата и время добавления в формате ISO 8601 (`YYYY-MM-DDTHH:MM:SS`) |

## Пример

```json
[
  {
    "vk_id": 123456789,
    "first_name": "Анна",
    "last_name": "Смирнова",
    "profile_url": "https://vk.com/id123456789",
    "photos": [
      "photo123456789_987654321",
      "photo123456789_987654322",
      "photo123456789_987654323"
    ],
    "saved_at": "2025-04-05T14:30:00"
  }
]
